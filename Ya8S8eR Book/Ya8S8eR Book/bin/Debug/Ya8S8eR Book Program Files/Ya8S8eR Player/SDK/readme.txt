                  KMP용 DSP 플러그인...

배포되는 플러그인은 델파이와 비주얼C용 2개 입니다.
다른 언어도 형식만 맞게 해주면 될것입니다. 
플러그인의 형식은 윈앰프의 DSP플러그인과 거의 같습니다.
음성의 경우는 100%같고, 영상도 거의 비슷합니다.
델파이는 KMPPlgIn.pas를 사용하면 되고 비주얼 C는 KMPPlgIn.h파일을
사용하면 됩니다. 예제는 음성의 경우는 잡음을 추가하는 기능과, 영상의
경우는 가로/세로 스캔라인을 표시하는 기능과 ColorProcess인터페이스를
이용한 시계입니다.
비주얼 C의 경우는 램덤함수를 구현할수가 없어서...(실력이 모자란 관계로)
잡음추가에 문제가 있습니다. 알아서 고쳐써세요...
예제를 보면 쉬게 이해가 될것입니다.
모두 DLL형식으로 작성하면 되구요...
특정 함수를 노출(Export)시키면됩니다.
호출규약은 윈앰프가 하는대로 cdecl(C언어 기본)입니다.
글고 파일의 이름은 DSP_*.dll이어야 합니다.

그럼... 플러그인 기능으로 자신만의 KMP를 만드시기 바람니다.
